# BeyogluV2 Backend

### Çalıştırma Gereksinimleri
 - python 3.8.x ve üzeri

### İlk kurulum
```bash
    pip install -r requirements.txt
    python manage.py makemigrations
    python manage.py migrate
```

### Çalıştırma
    python manage.py runserver 0.0.0.0:8000
